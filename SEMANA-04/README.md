# Proyecto Web Profesional con Bootstrap 5

Este proyecto presenta una página web moderna, responsiva y estilizada utilizando **Bootstrap 5** y **CSS3 avanzado**.

## Tecnologías
- HTML5
- CSS3
- Bootstrap 5
- GitHub Pages

## Funcionalidades
- Navbar responsiva
- Diseño Grid y Flexbox
- Animaciones CSS
- Formularios adaptables
- Componentes interactivos (Collapse)
- Publicación en GitHub Pages
