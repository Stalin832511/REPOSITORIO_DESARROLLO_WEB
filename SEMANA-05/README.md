# Galería Interactiva de Imágenes - Semana 5

Este proyecto consiste en una galería de imágenes dinámica creada con HTML, CSS y JavaScript.

## Funcionalidades

- Agregar imágenes mediante URL.
- Seleccionar una imagen con clic.
- Resaltar la imagen seleccionada.
- Eliminar la imagen seleccionada.
- Atajos de teclado:
  - Enter: Agregar imagen.
  - Delete: Eliminar imagen seleccionada.
